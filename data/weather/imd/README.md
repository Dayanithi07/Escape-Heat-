## Description
IMD provides gridded daily temperature (1° x 1° and 0.5° x 0.5° resolution) and rainfall (0.25° x 0.25° resolution) datasets for the Indian subcontinent. Because these datasets require formal registration, verification, or purchase for official use, they cannot be programmatically downloaded.

## Intended Usage
Historical weather gridded baseline calculations and validation of local heat trends for urban environments in India.
